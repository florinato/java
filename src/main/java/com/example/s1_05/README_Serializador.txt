
# Instrucciones para Ejecutar Serializador

## Descripción
Este programa realiza la serialización y deserialización de un objeto de la clase `Persona` en Java. 

## Compilación
Para compilar el programa, asegúrate de que los archivos `Serializador.java` y `Persona.java` estén en el mismo directorio. Luego ejecuta el siguiente comando en la consola:

```
javac Serializador.java Persona.java
```

## Ejecución
Después de compilar, ejecuta el programa utilizando el siguiente comando:

```
java Serializador
```

El programa serializará y deserializará un objeto `Persona` y mostrará los resultados en la consola.

## Notas Adicionales
- Asegúrate de tener Java instalado en tu sistema.
- Este programa debe ejecutarse desde el directorio donde se encuentran los archivos `.java`.
- Si modificas el código, recuerda volver a compilar antes de ejecutar.
