
# README - Compilación de AESUtil

## Descripción
Este documento provee instrucciones para compilar la clase AESUtil, que es parte
de un programa Java para encriptación AES.

## Requisitos
- Java JDK 17 o superior.

## Estructura de Archivos
Asegúrate de tener la siguiente estructura de archivos en tu proyecto:
- `src/main/java/com/example/s1_05/n3/`: Contiene los archivos de código fuente.

## Compilación de AESUtil

Para compilar la clase AESUtil, sigue estos pasos:

1. **Compilar AESUtil**
   Navega al directorio raíz de tu proyecto y ejecuta el siguiente comando:
   ```
   javac src/main/java/com/example/s1_05/n3/AESUtil.java -d bin
   ```
   Esto compilará `AESUtil.java` y colocará el archivo `.class` resultante en el directorio `bin`.

.

## Ejecución de AESUtil

Después de compilar la clase, puedes ejecutar el programa con el siguiente comando:

```
java -cp bin com.example.s1_05.n3.AESUtil
```
