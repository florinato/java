
# README

## Descripción

Este programa se encarga de la serialización y deserialización de objetos en Java, utilizando el paquete `com.example.s1_05.n1.e5`.

### Compilación

1. Abre una consola o terminal.
2. Navega hasta el directorio `c:\Users\oscar\Desktop\Nueva carpeta\demo`.
3. Ejecuta el siguiente comando para compilar el programa:

   ```bash
   javac -d src\main\java\com\example\s1_05\n1\e5\Serializador.java src\main\java\com\example\s1_05\n1\e5\Persona.java
   ```

   Esto compilará los archivos `.java` del paquete `com.example.s1_05.n1.e5` y generará archivos `.class` correspondientes.

### Ejecución

Una vez compilado, ejecuta el programa con el siguiente comando:

```bash
java -cp src\main\java com.example.s1_05.n1.e5.Serializador

```
